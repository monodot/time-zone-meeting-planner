Find the perfect meeting time across timezones. Visualize working hours and find overlapping time slots for international meetings.

## Features

- Select multiple timezones
- Visual 24-hour timeline with working hours highlighted
- Interactive time slider to explore meeting times
- 24-hour clock format
- Responsive design

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Deploy to GitHub Pages

1. Update `vite.config.js` with your repository name
2. Run: `npm run deploy`
